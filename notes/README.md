# hanime-sama

## Objectif
Site pour explorer des animes, manhwas, mangas avec navigation simple.

## Fichiers principaux
- index.html : accueil
- /anime/, /manwha/, /manga/ : sections dédiées
- /code/ : accès par code
- /data/ : données JSON
- /css/ : styles séparés ordi/tel
- /js/ : scripts par type
- /notes/ : docs de projet
